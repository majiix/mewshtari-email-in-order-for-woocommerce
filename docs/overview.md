# Project Overview

## Description
Adds a WYSIWYG panel on the WooCommerce order edit screen to send custom emails to customers and update order status.

## Tech Stack
- WordPress 6.x+
- WooCommerce
- PHP 7.4+
- JavaScript (jQuery)

## Verification Commands
- PHP Syntax Check: `php -l mewshtari-email-in-order-for-woocommerce.php`
